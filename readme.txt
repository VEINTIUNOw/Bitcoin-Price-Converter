A simple plugin to convert Bitcoin prices to various fiat currencies with real-time exchange rates.

== Description ==

Bitcoin Price Converter is a lightweight WordPress plugin that allows you to convert Bitcoin (BTC) to various fiat currencies. The plugin provides:

* Real-time Bitcoin price conversion
* Support for multiple currencies (USD, EUR, GBP, etc.)
* Admin settings panel for configuration
* Shortcode support for easy embedding
* Responsive design that works on all devices

Perfect for cryptocurrency websites, financial blogs, or any site that needs Bitcoin price conversion functionality.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/bitcoin-price-converter` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Settings > Bitcoin Converter to configure the plugin
4. Use the shortcode [bitcoin_converter] to display the converter on your pages/posts

== Frequently Asked Questions ==

= What currencies are supported? =

The plugin currently supports USD, EUR, and GBP. More currencies will be added in future updates.

= How often are the prices updated? =

The plugin fetches real-time prices from cryptocurrency APIs whenever a conversion is requested.

= Can I customize the appearance? =

Yes, the plugin includes CSS classes that can be styled to match your theme.

= Is there a shortcode available? =

Yes, you can use [bitcoin_converter] shortcode to display the converter anywhere on your site.
